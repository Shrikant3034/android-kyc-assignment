Overview
This project is a Digital Banking KYC Android application developed as part of the Android Developer assignment.
The app allows a Relationship Manager to manage customers and perform the KYC (Know Your Customer) process through a simple and user-friendly interface.

Features
- Login Screen
- Customer List
- Search Customers
- View Customer Details
- KYC Verification
- Upload Customer Documents
- Clean and Responsive UI
- Easy Navigation

Tech Stack
- Kotlin
- Android Studio
- XML
- Material Design
- RecyclerView
- MVVM Architecture

 Project Structure
app
├── activities
├── adapters
├── models
├── ui
├── utils
└── res
